《荒岛 X 爱 X 求生》：重生之我在荒岛过神仙生活沙盒SLG/动态官中 ProjectIsland

《荒岛 X 爱 X 求生》是由“无人岛观光促进会“开发，并由Alice Publication发行的一款休闲策略养成游戏。如果你是一名H游戏爱好者，对于发行商应该不会陌生到哪里去了，旗下发行游戏基本都是精品的水准，《荒岛 X 爱 X 求生》也是继去年发行商（Or制作组）宣布解散前发行游戏《剑侠．风尘恋曲》时隔一年后的新作。本次就简单介绍一下这次新作的体验感受吧~

H游戏的背景故事离谱也不是一天两天了，但是可能因为我最近在一些荒岛求生类的视频缘故，不知道为何看到这个同样将背景故事设定在荒岛上的H游戏就有点发笑。在游戏的背景故事中，因为背负巨额债款，我们的主角逃入了邮轮中，在大海上行驶时，恰好遭遇了船舱进水等问题，于是乎我们的主角流落到了荒岛，而这正好契合它的梦想——找一个没人认识的地方不用背负债务重新开始生活。

和其他同类游戏不同的是，这款游戏在一开始就给玩家送了一波福利，救助主角的正是独自生活的荒岛的少女，而救助主角的方式不是进行传统意义的人工呼吸，只能说玩家懂得都懂就不展开太细的描述，不过这对于H游戏来说再不合理都变得合理了。因为设定是荒岛求生，所以游戏内自然也就包括了生存元素，我们在每天都需要关注自己的饱食度和饥渴值，这两项数值在每天一开始都会进行降低，也就是说我们每天都需要摄入一定的食物和水分。

游戏的具体玩法简单来说就是，在每个地图当中收集资源去制作物品、建造设施，例如初始地图内分布有浆果、树枝、石头、还有树叶，在一开始我们搭建临时住所就需要消耗这些资源，而这就需要我们在不同的地图内去收集上述资源。并且如蘑菇等资源在白天内是不能进行收集的，必须在游戏内时间推进到夜晚时才可以收集，一些特殊资源甚至需要我们在白天选择探索才可以获取。

游戏内最玩家可探索的区域和收集的资源十分有限，我们只能通过收集当下已有的资源→制造火把、绳索等物品解锁新地图→重复上述过程直到解锁全部地图和女主角。需要特别说明一下的是，游戏的内解锁女主角的方式比较离谱，在荒岛内设置了一处特殊场所，通过收集能量石可以解锁雷击天气兵器的使用权限，而这项能力则是用在了击沉飞机上面，每当有飞机飞过时玩家可以选择击沉飞机，而女主角也会解锁一位新的。

■布景简介

穷途末路的您决议跟船出海以归还巨额债权，却正在途中遭受海易，船只也因此淹没。
所幸劫后余生，您乘着洋流被推上一座奥秘的岛屿。
正在岛上您拆建起营天，睁开了新的人死。
时期相逢了本地的住人娜迪、援救其他罹难者，于一次次的变乱中取各个夥陪减深相互豪情；
而借由她们的协助，强大开展基天、探究小岛没有为人知的奥秘地区，并从中发明更多的宝躲取机密。

至于H环节，主要是依靠传统的好感度解锁CG内容和进行交易，我们可以把H环节当作交易一环去进行交易。总的来说，在游戏前期本作就H上面的内容是不足的，而随着流程的推进后续的H内容才可以说是量大管饱，但实际上，游戏内对于H的刻画还是H外的玩法都是比较浅层的，《荒岛 X 爱 X 求生》只能算是比较平庸的小H游，要求不高的玩家还是可以试试。
